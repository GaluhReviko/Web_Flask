Website Analisis Sentimen Netizen terhadap Program IKN

Website ini dikembangkan untuk menganalisis sentimen masyarakat terhadap kebijakan pemindahan Ibu Kota Negara (IKN), berdasarkan opini yang diambil dari media sosial Twitter. Dengan menggunakan algoritma Naive Bayes, sistem ini dapat mengelompokkan komentar netizen ke dalam lima kategori: sangat mendukung, mendukung, netral, tidak mendukung, dan sangat tidak mendukung.

Fitur-fitur utama dalam website ini meliputi unggah data CSV, pemrosesan otomatis (pembersihan, normalisasi, stemming, dan terjemahan), pelabelan sentimen, serta evaluasi kinerja model menggunakan akurasi dan nilai F1-score.
